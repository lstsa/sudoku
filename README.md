# sudoku

## o que é?
Um gerador/solucionador de sudoku com o propósito técnico de repassar o básico de python.

## o que faz?
O projeto gera um puzzle sudoku para que o usuário o preencha pelo próprio terminal, utiliza de seed ( data ) para gerar puzzles diários, suporta 3 dificuldades, realiza salvamento automático após cada jogada.

## como roda?

```bash
python main.py
```

## testes automáticos:
```bash
pytest test_validator.py -v
```
